## description
Task master mobile application with three pages:
1. home page with two buttons and image:
- Add new Task Button
- All Tasks Button
2. AddNewTask page:
- Add task button that show submitted label on the page when clicked.
- back button that back to home page.
- task description
- task title
3. AllTasks page:
- Image
- back button that back to home page.
